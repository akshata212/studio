# **App Name**: PixelStudio

## Core Features:

- Dynamic Portfolio & Gallery: Showcase high-quality photography work through a sleek, dynamic gallery, with image metadata managed in Firestore and assets stored in Firebase Storage.
- Online Booking System: A user-friendly form for clients to submit booking inquiries, capturing details like name, contact information, event type, date, and message, with data securely saved to Firestore.
- Admin Booking & Inquiry Dashboard: A protected administrative panel with Firebase Authentication for studio staff to view and manage incoming booking requests and contact form submissions in real-time.
- Studio Information Pages: Responsive marketing pages including Home (with hero slider, services preview, testimonials), Services, and Contact, all designed with a premium aesthetic.
- Content Description Tool: An AI-powered tool within the admin dashboard to generate elegant captions or descriptions for portfolio images and service offerings.
- Direct Client Communication: Integrate a floating WhatsApp button for instant client communication and embed Google Maps for easy studio location lookup.

## Style Guidelines:

- Primary color: Vibrant Teal (#00BFA5) for a fresh, modern, and attractive look. (HSL: H=170, S=100%, L=37%)
- Background color: Clean, light grey (#F0F2F5) to make colorful elements pop and maintain a bright, inviting feel. (HSL: H=210, S=10%, L=95%)
- Accent color: Energetic Coral (#FF7043) for calls-to-action and highlights, providing a vivid contrast. (HSL: H=16, S=100%, L=63%)
- Secondary color: Calming Deep Purple (#673AB7) for visual balance and depth. (HSL: H=260, S=60%, L=47%)
- Headline font: 'Montserrat' (sans-serif) for a strong, modern, and attractive presence. Body font: 'Open Sans' (sans-serif) for excellent readability and a contemporary feel.
- Use modern, geometric icons, potentially with duo-tone or vibrant filled styles, that complement the colorful and attractive theme. Ensure consistency in shape and boldness.
- Implement dynamic and engaging layouts with spacious sections. Maintain a full-width hero section with an automatic image slider. Utilize glassmorphism cards with vibrant backdrops, soft shadows, and consistently rounded corners for a modern, attractive aesthetic.
- Incorporate smooth, noticeable animations and motions throughout the interface, including dynamic fade-ins, scroll-triggered effects, subtle hover interactions, and engaging transitions for image galleries. Ensure a clear and delightful success animation for form submissions.
